# Deep SEO Optimization

## When to use this sub-skill
- When structuring the HTML of a new web application or landing page.
- When auditing a page for Google ranking performance.

## Workflow
1.  **Technical HTML**: Ensure `<h1>` to `<h6>` tags are strictly hierarchical.
2.  **Meta & Schema**: Validate Open Graph tags, canonical URLs, and structured data (JSON-LD) for rich Google results.
3.  **Performance Context**: Ensure images are optimized (WebP) and lazy-loaded to maximize Core Web Vitals.

## Instructions
- Web-apps must still be discoverable. Treat every route as an SEO opportunity.
- Ensure all texts mapped from databases or APIs have SEO weight.
