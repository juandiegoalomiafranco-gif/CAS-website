# Frontend Roadmap Skill

## When to use this skill
- When starting a new web project or choosing a tech stack.
- When you need to decide which framework or library to use.
- When organizing the learning or execution path of a complex site.

## Workflow
1.  **Select the Stage**: Identify where the project stands (Basics, Frameworks, AI, or Portfolio).
2.  **Verify the Foundations**: Use HTML, CSS, and vanilla JS for pure, high-performance base code.
3.  **Upgrade Strategically**: If complexity increases, select from React, Angular, or Vue as suggested by the roadmap.
4.  **Integrate AI Tools**: Use AI (e.g., this agent, Copilot) to accelerate the building of specific sections.

## Roadmap Categories
- **Basics**: HTML5 (Semantic), CSS3 (Flex/Grid), JavaScript (Modern).
- **Styling Libraries**: Bootstrap, Tailwind CSS, SASS/SCSS.
- **VCS**: Git/GitHub for collaborative or tracked development.
- **Portfolio & Network**: Building a showcase-ready site.

## Instructions
- Ensure all basic HTML is **Semantic** (using `<article>`, `<section>`, `<nav>`).
- Use Modern CSS (Grid/Flexbox) exclusively—NO legacy table layouts.
- Keep the roadmap's "Industry Standards" in mind for every project.
