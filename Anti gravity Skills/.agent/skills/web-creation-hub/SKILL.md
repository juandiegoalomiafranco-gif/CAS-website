# Web Creation Hub

## When to use this skill
- When starting or improving a web project (e.g., Dr. Agudelo's site).
- When looking for the most efficient way to implement a feature.
- When verifying the quality and structure of a frontend implementation.

## Workflow
1.  **Consult the Roadmap**: Determine the current stage of development and what technologies to prioritize (following-frontend-roadmap).
2.  **Use Optimized Snippets**: Implement high-quality, performant logic and animations using the verified "30 seconds" collection (using-30s-snippets).
3.  **Verify with Checklist**: Before finalizing any section, run through the comprehensive web development checklist to ensure nothing was missed (web-development-checklisting).

## Instructions
- Ensure all designs are "Premium" and follow the `designing-premium-brands` guidelines.
- Leverage the sub-skills in this folder to maintain high-quality code.
- Prioritize CSS-only animations for performance, and JS-only for complex interactivity.
