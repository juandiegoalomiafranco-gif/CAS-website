# Advanced Animations (WOW Factor)

## When to use this sub-skill
- When standard CSS transitions are not enough.
- To create staggered load effects, scroll-based timelines, and liquid interactions.

## Workflow
1.  **Assess Need**: Use JS animations only for complex choreographies; stick to CSS for simple hovers.
2.  **Implement**: Utilize high-performance libraries or `requestAnimationFrame` logic to ensure 60fps.
3.  **Optimize**: Ensure animations pause when not in the viewport or when the user prefers reduced motion (`prefers-reduced-motion`).

## Instructions
- Focus on micro-interactions that make the app feel "alive".
- Avoid heavy blocking animations that disrupt the user flow.
