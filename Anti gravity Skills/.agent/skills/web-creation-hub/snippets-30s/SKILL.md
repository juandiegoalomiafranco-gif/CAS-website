# 30-Seconds Snippet System

## When to use this skill
- When you need a quick, performant solution for a common programming problem.
- When creating UI components (animations, transitions, data formatting).
- When you want "Premium" interactivity without the overhead of heavy libraries.

## Workflow
1.  **Identify the need**: (e.g., "shuffling a list of surgical procedures" or "adding a scroll-to-top button").
2.  **Pull from 30s Collection**: Search for the most modern implementation in the local knowledge or repo.
3.  **Integrate and Test**: Place the snippet in the appropriate file (JS or CSS) and ensure it works with existing code.

## Key Snippet Categories
- **JavaScript Fundamentals**: Manipulating arrays, objects, and strings.
- **Interactivity & DOM**: Handling events, mouse coordinates, and scroll behavior.
- **CSS Transitions**: Pure CSS animations and styling tricks.

## Instructions
- Always prioritize **readability** and **performance**.
- Don't reinvent the wheel; look for a snippet FIRST before writing custom logic.
- Ensure all JS is **ES6+** (const/let, arrow functions, destructuring).
