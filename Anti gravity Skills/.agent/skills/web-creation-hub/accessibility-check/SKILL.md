# Accessibility & WCAG Audit

## When to use this sub-skill
- To ensure contrast ratios hold up, especially in "Dark Premium" themes.
- When building forms, modals, and interactive app components.

## Workflow
1.  **Color Contrast**: Ensure text against dark backgrounds passes ADA/WCAG AA contrast ratios (minimum 4.5:1).
2.  **Keyboard Navigation**: Ensure all interactive elements can be focused via the Tab key.
3.  **ARIA Labels**: Use `aria-label` and `role` attributes for custom app components.

## Instructions
- A premium site is an accessible site. Do not skip `alt` tags or screen-reader context.
