# Marketing & Growth Strategy

## When to use this skill
- User asks for conversion rate optimization (CRO) on a page.
- User needs copywriting for landing pages, emails, or social media.
- User wants SEO audits, content strategy, or technical SEO improvements.
- User is asking for growth ideas, lead magnets, pricing strategies, or retention methods.

## Workflow
1. **Analyze Context**: Ask the user for the target audience, product value proposition, and current marketing bottleneck (or review existing marketing context files).
2. **Determine Sub-Domain**:
   - **CRO**: Evaluate landing pages, signup flows, forms, and trust signals.
   - **Copywriting**: Review tone, clarity, headlines, and calls to action (CTAs).
   - **SEO & Discovery**: Check schema markup, site architecture, and content structure.
   - **Strategy**: Brainstorm lead magnets, retention tactics, or campaign launches.
3. **Draft Recommendations**: Present a structured list of actionable improvements prioritizing high-ROI changes.
4. **Implement**: Apply approved changes to the user's website codebase, content files, or configuration.

## Instructions
- **Psychology-First**: Focus heavily on user psychology, mental models, and addressing objections before they arise.
- **Copywriting**: Use strong, benefit-driven headlines. Cut fluff. Optimize for readability and scanning.
- **CRO Design**: Minimize friction in forms. Ensure clear, contrasting CTAs. Include robust social proof (testimonials, trust badges).
- **SEO**: Prioritize semantic HTML, fast load times, and structured data (Schema.org).
- **Data & Tracking**: When applicable, suggest proper event tracking (e.g., GA4, Pixel) for key conversion events.
